# QuizRoom — Real-time Multiplayer Quiz Platform

## Быстрый старт

### 1. Установка зависимостей
```bash
npm install
```

### 2. Запуск сервера
```bash
npm start
```
Или для разработки с автоперезагрузкой:
```bash
npm run dev
```

### 3. Откройте браузер
```
http://localhost:3000
```

---

## Как пользоваться

### Хост (создатель квиза)
1. Нажмите **«Создать квиз»**
2. Введите название, своё имя, настройте время на вопрос
3. Добавьте вопросы — введите текст, 4 варианта ответа, пометьте правильный (зелёная галочка)
4. Нажмите **«Запустить»** — получите код комнаты
5. Поделитесь кодом с игроками
6. Нажмите **«Начать игру»** когда все подключились

### Игрок
1. Нажмите **«Войти по коду»**
2. Введите имя и код комнаты от хоста
3. Ожидайте начала игры
4. Отвечайте быстрее — очки зависят от скорости ответа!

---

## Система очков
- Правильный ответ: **500 очков**
- Бонус за скорость: до **+500 очков** (зависит от того, насколько быстро ответили)

## Деплой в облако

### Railway (рекомендуется)
1. Зарегистрируйтесь на railway.app
2. Создайте новый проект → Deploy from GitHub
3. Загрузите эти файлы в репозиторий
4. Railway автоматически запустит `npm start`

### Render
1. render.com → New Web Service
2. Укажите команду: `npm start`
3. Порт: 3000

### Переменные окружения
```
PORT=3000   (по умолчанию)
```

- **Backend**: Node.js, Express, Socket.io 4.x
- **Frontend**: Vanilla HTML/CSS/JS + Socket.io Client
- **Шрифты**: Syne + DM Sans (Google Fonts)
- **Деплой**: любой Node.js хостинг
